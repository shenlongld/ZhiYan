1.vectorsize.py {
    读取json，做向量化处理
    对于学校-学院，统计其入营的人，不入营的人（键要对，一学校-学院只有一个键）
    对于人（发帖者）：统计其拥有的竞赛条目，论文条目，院校层级（中九，985，211等），成绩百分比，入营了哪个，不入营哪个；
}

2.value.py : {

基于上述向量化的信息

对于同一个院校的夏令营，入营的，对其条目记正，否则记负，最后对：院校层级，论文，竞赛进行归100化处理

所有的含金量评定都通过这个

示例：院校A: 30个样本, 17个入营，13个未入营，对入营条目，它拥有的每一个竞赛，论文，院校层级（985，中九等）+1，总分则*（13/30）来进行难度约束，对不入营的则-1, 总分*（17/30） 然后对每个条目进行全局归1化

}

3.train.py {
    基于上述的value，综合学生的竞赛条目，论文条目，院校层级（中九，985，211等）value评定的结果，成绩百分比。使用tinyMLP计算每个院校在这四个层面的权重。并给出院校的加权总分bar，推荐报考分
}

4.test.py {
    用户输入结构化的文本，对入营gap分从高到低排个序
}
